# Task manager

Ce system indique les taches a realiser et permet de les organiser par priorite et statut.
Il est construit avec React et TypeScript pour une experience utilisateur fluide et interactive.

## Contenu

- Liste des taches avec details
- Formulaire d'ajout/edition de taches
- Suppression de taches
- Stockage local des taches

## Installation

1. Cloner le depot: `git clone https://github.com/votre-utilisateur/task-manager.git`
2. Installer les dependances: `npm install`
3. Demarrer l'application: `npm run build`
4. Utiliser le localhost: `http://localhost:5173` pour acceder a l'application dans le navigateur (si le port est déjà utilisé, le port suivant sera pris en compte).

## Fonctionnalites non implementees

- Filtrage et tri des taches (manque de temps + non-cohérent avec la structure de ma liste (organisation en carte))
- Date d'echeance et rappels (manque de temps)
